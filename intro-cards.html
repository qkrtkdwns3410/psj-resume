주요 프로젝트

  • 신한카드 프로젝트 - Thymeleaf 기반 서버사이드 렌더링과 Spring Boot 백엔드를 결합하여 약 한 달 만에 사용자 서비스와 어드민 시스템을 완성. 확장성 있는 구조 설계를 주도하여 향후 아이템베이 메인 플랫폼에도 쉽게 이식할 수 있도록 설계.
  • 아이템베이 플랫폼 - 안정된 서비스에서 하루에도 여러 차례 배포가 필요한 환경에서 핵심 기능 우선순위를 빠르게 재정리하고, 긴급 배포 과정에서도 서비스 안정성을 유지하며 일정에 맞춰 개발을 완수.
  • 개인 기술 블로그 - 학습 내용을 체계적으로 정리·공유하여 검색 의존도를 줄이고 개인 기술 레퍼런스를 구축. 문제 해결 과정과 코드를 다른 개발자도 이해할 수 있도록 체계적으로 기록.

경력 사항

  • 아이템베이 - 풀스택 개발자 (2023년 ~ 현재)
